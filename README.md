### What is convolution?

Convolution is a process used in CNNs to detect important features in images such as edges, textures, shapes, and patterns.  
Small filters move across the image and help the model learn useful visual information.


### Why is pooling used?

Pooling is used to reduce the size of feature maps while keeping important information.  
It helps reduce computation, speeds up training, and prevents overfitting.


### Why is ReLU commonly used in CNNs?

ReLU (Rectified Linear Unit) is an activation function that converts negative values to zero and keeps positive values unchanged.  
It helps the model learn faster and improves the performance of the neural network.


### Why are CNNs better than regular feed-forward networks for image data?

CNNs are specially designed for image processing.  
They automatically learn visual features from images using convolution layers and require fewer parameters compared to regular feed-forward neural networks.  
This makes CNNs more efficient and accurate
